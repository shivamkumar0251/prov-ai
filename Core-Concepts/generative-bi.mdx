title: "Generative BI"
description: "The evolution from traditional static dashboards to dynamic AI-driven insights."
---

**Generative BI (Business Intelligence)** is a modern approach to data analytics where users interact with data using natural language instead of pointing, clicking, and dragging in traditional dashboarding software or writing SQL manually.

By integrating LLMs, the data warehouse, and a semantic layer, Prov AI generates answers, visualizations, and data narratives instantly on the fly.

---

## Traditional BI vs. Generative BI

<Frame>
| Feature | Traditional BI | Generative BI |
| :--- | :--- | :--- |
| **Primary Interface** | Dashboard-first | AI-first (Chat-first) |
| **Technical Skill** | Requires SQL & Tool Training | Accessible via Natural Language |
| **Report Type** | Static, pre-built reports | Dynamic, real-time answers |
| **Exploration** | Manual point-and-click slicing | Conversational follow-ups |
| **Time to Insight** | Days or Weeks | Seconds |
</Frame>

---

## The Generation Lifecycle

When you ask a question, Prov AI "generates" much more than just numbers:

1. **Generates SQL**: Code written for the specific database dialect.
2. **Generates Visualizations**: Automatically rendering the data using the most effective chart type.
3. **Generates Logic Summaries**: Explaining *how* the data was calculated to prove accuracy.
4. **Generates Insights**: Writing out natural language summaries detailing anomalies, trends, or notable data points.

## 4. Prov Engine

Prov Engine is the semantic runtime that processes SQL, rewrites logic, and transpiles queries for multiple dialects.

<img src="/images/prov-engine.png" alt="Prov Engine showing SQL processor, SQL transpiler, metastore, and multi-dialect connectors" />

Core capabilities:

- SQL planning and SQL rewriting on semantic context
- Dialect transpilation for multiple warehouse engines
- Metadata-driven query behavior through metastore integration
- Consistent query experience across heterogeneous data stacks

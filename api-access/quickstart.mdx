Learn how to authenticate and make your first natural language query using the Prov AI API.

---

## 1. Prerequisites

- An active **Prov AI Account**.
- A **Project API Key** (Generate this in **Settings > API Access > Project Keys**).
- Your **Project ID** (Visible in the browser URL or Project Settings).

---

## 2. Make Your First Request

Use the following `curl` command to ask a question via the API. Replace `<PROV_API_KEY>` and `<PROJECT_ID>` with your credentials.

```bash
curl --request POST \
  --url https://api.prov.ai/api/v1/projects/<PROJECT_ID>/ask \
  --header 'Authorization: Bearer <PROV_API_KEY>' \
  --header 'Content-Type: application/json' \
  --data '{
    "question": "What is the total revenue for the last 30 days?"
  }'
```

---

## 3. Understanding the Response

The response includes the reasoning, the generated SQL, and the data result (up to 50 rows).

| Field | Description |
| :--- | :--- |
| **sql** | The literal SQL generated by the Prov Engine for your database. |
| **reasoning** | Natural language explanation of how the AI understood the question. |
| **result** | An array of objects representing the data rows. |
| **columns** | Metadata for each column in the result set (name, type). |

Welcome to the Prov AI API documentation. You can use our API to embed chat interfaces, execute queries programmatically, and manage semantic models.

## Authentication

All API endpoints are authenticated using Bearer tokens.

```bash
curl -X GET "https://api.prov.ai/v1/..." \
  -H "Authorization: Bearer YOUR_API_KEY"
```

Calculated fields empower you to generate custom columns by employing expressions to compute values. This is similar to using formulas in spreadsheet software like Excel or Google Sheets.

---

## Create a Calculated Field

<Steps>
  1. **Select Table**: Go to the diagram of the model where you want to add a field.
  2. **Add Field**: Click the **+** icon next to the **Calculated Fields** section.
  3. **Name it**: Enter a descriptive name for your new column.
  4. **Choose Function**: Select the desired function from the list of supported expressions.
  5. **Select Column**: Choose the column(s) to apply the function to.
     - *Tip*: You can reference columns from other models if a relationship is already defined.
  6. **Save**: Click **Save** to create the field.
</Steps>

---

## Managing Calculated Fields

### Update a Field
1. Click the **More icon (⋮)** next to the calculated field on the model diagram.
2. Choose **Edit**. 
3. Modify the name or expression as needed and click save.

### Delete a Field
1. Click the **More icon (⋮)** next to the calculated field.
2. Choose **Delete**.

---

## Important: Deploy Changes

All modifications in the Modeling page must be synchronized to take effect.

- **Undeployed Changes**: A message will appear on the navigation bar if there are pending updates.
- **Sync**: Click the **Deploy** button to sync your changes. Once completed, the status will show as **Synced**.

Prov AI is built with privacy-first principles. We prioritize the security of your data and the confidentiality of your team's interactions.

---

## Our Privacy Commitments

| Commitment | Description |
| :--- | :--- |
| **Data Isolation** | Your data is never mixed with other organizations or used for public AI training. |
| **Local Processing**| The Prov Engine runs within your infrastructure (Self-Hosted) or isolated cloud environments. |
| **Encryption** | All data in transit is encrypted using TLS 1.3, and data at rest is protected with AES-256. |
| **GDPR & CCPA** | We comply with global data protection standards for user privacy and data access. |

---

## LLM Interaction

Prov AI acts as a middleware between your database and the LLM provider.

- **No Data Storage**: We do not store your raw database records. Only model metadata (schema names, descriptions) is sent to the LLM.
- **Opt-out Training**: We ensure our agreements with LLM providers (e.g., OpenAI, Anthropic) do not allow your project metadata to be used for model training.

<Note>
  If you are using **Bring Your Own LLM**, your data privacy is also governed by your organization's direct agreement with the LLM provider (e.g., Azure or AWS).
</Note>

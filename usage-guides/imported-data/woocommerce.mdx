Analyze your WooCommerce store performance in Prov AI Cloud using our boilerplate. This template provides a structured framework for orders, products, and customer analytics.

---

## Prerequisites

- **WooCommerce Account**.
- **REST API Key**: Generate a key and secret with **Read** permissions in your WordPress admin panel.

---

## Setup Steps

1. Select **WooCommerce** in the "Start with a data boilerplate" section.
2. Provide your **API Key**, **API Secret**, and **Shop Name** (e.g., `example.com`).
3. Select a **Start Date** for data replication.
4. Click **Start Syncing**.

---

## Customization

### Adding Custom Columns
1. Select a table in the **Modeling** page.
2. Click the **⋮** menu and select **Update Columns**.
3. Select the additional fields you need and hit submit.

### Sync Schedule
Adjust how frequently Prov AI pulls data from WooCommerce in **Settings > Data Connection**.

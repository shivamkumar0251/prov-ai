## UI Components

### 1. Query Input Box
- Accepts free-form text or voice input.
- Supports intelligent autocomplete suggestions based on your Semantic Layer metrics and dimensions.

### 2. Response Panel
This dynamic area displays the results of your query:
- Interactive charts and detailed data tables.
- Narrative summary insights explaining the trend.
- A toggle to view the generated SQL.

### 3. Chat History
- Maintains a thread of your previous queries.
- Enables a seamless conversational flow for deep-dive analysis.

### 4. Suggested Prompts
To help you get started, the interface provides dynamic examples:
- *“Sales by region”*
- *“Top products last month”*

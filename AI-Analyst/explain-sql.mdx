## Example

**User Query:**
> *"Revenue by region"*

**Generated SQL:**
The system generates a highly-optimized query specifically structured for your data warehouse dialect (e.g., Snowflake, BigQuery):

```sql
SELECT 
    region, 
    SUM(revenue) AS total_revenue
FROM sales_data
GROUP BY region
ORDER BY total_revenue DESC;
```

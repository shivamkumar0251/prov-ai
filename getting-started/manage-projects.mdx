---

Once a project is created, you can set up a data source connection and start asking questions. 

<Note>
  **Data Sync**: Synchronization can take some time. Please wait for the initial sync to complete to ensure accurate results.
</Note>

### Supported Data Sources

- **Cloud Data Warehouses**: BigQuery, Snowflake, Redshift, Athena, Databricks.
- **Relational Databases**: PostgreSQL, MySQL, Oracle, SQL Server.
- **OLAP & Big Data**: ClickHouse, Trino, Spark (EMR/Athena).
- **Files**: CSV Upload.

## Access Control

Manage user permissions within your project. **Project Owners** have full control. Prov AI supports three roles:
1. **Project Owner**: Full access to all settings and data.
2. **Project Contributor**: Can edit models and instructions.
3. **Project Viewer**: Can only ask questions and view results.

### Add Member
To add a member, they must first be invited to your organization. Select the member and assign their project role.

<img src="/images/manage-project-5.png" alt="Add Member" />

---

### Update or Remove Members
- **Update Permission**: Use the **Permission** dropdown menu. Note: You cannot modify your own role.
- **Remove Member**: Revokes all project access. Note: Organization Admins cannot be removed from projects.

## Danger Zone

### Reset Project
<Warning>
  **Reset Project**: This action is **permanent**. It deletes all settings, metadata (Modeling page), and question history.
</Warning>

<img src="/images/manage-project-10.png" alt="Reset Project" />

### Delete Project
<Danger>
  **Delete Project**: Once deleted, all questions and results are lost forever. This action cannot be undone.
</Danger>

<img src="/images/manage-project-11.png" alt="Delete Project" />

<Note>
  You cannot delete a project if it is your last accessible project in the organization.
</Note>
